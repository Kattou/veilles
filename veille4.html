L'actualité sur les langages informatiques : C++ pourrait dépasser Java et monter sur le podium

LE CLASSEMENT DES LANGAGES

Pour ce nouveau classement TIOBE de juin, le podium reste identique au mois dernier : Python domine le classement (hausse de l’indice de 0,35 %), suivi par C (-0,64 %), et Java (-1,07 %). Une perte de vitesse pour ce dernier, qui était encore à la première place en avril 2020. Parallèlement, ce déclin pourrait bien profiter à C++, 4e du classement, qui signe une belle croissance, à +2,26 %. LES AVANTAGES DU C++

La programmation de haut niveau et les performances sont les principales raisons pour lesquelles le C++ se porte si bien. Le langage C++ est révisé tous les trois ans. Ces révisions contiennent de nouvelles fonctionnalités révolutionnaires, qui placent le langage au même niveau que C# et Java, mais sans la pénalité de performance […], analyse Paul Jansen, CEO de TIOBE. Pour le mois de juin, voici le top 10 des langages de programmation selon l’index TIOBE : 1 Python 2 C 3 Java 4 C++ 5 C# 6 Visual Basic 7 JavaScript 8 SQL 9 Assembly language 10 Swift
